《寻龙诀》为什么好？因为买版权的姿势正确

suanwusi043677 2015-12-26 14:51:18 0℃ 独立鱼 评论

摘要：《鬼吹灯之寻龙诀》上映之后，首周末三天票房累积近六亿。在众多媒体网民口中，这样一个由 IP 改编的大电影被称为华语电影工业大片，让不少人重燃对国产大片的期望。笔者看过之后也觉得非常不错，不仅是内容不错，IP运作上也是相当出色！

《鬼吹灯之寻龙诀》上映之后，首周末三天票房累积近六亿。在众多媒体网民口中，这样一个由 IP 改编的大电影被称为华语电影工业大片，让不少人重燃对国产大片的期望。笔者看过之后也觉得非常不错，不仅是内容不错，IP运作上也是相当出色！

关于「鬼吹灯」系列

 

鬼吹灯系列是一个以盗墓探险为主题、具有独立世界观的小说，每一部都具有故事发展性，而且是系列小说，因此具备非常高的影视改编价值。据天下霸唱介绍称：《鬼吹灯》总共写了两部八本，第一部和第二部是分着卖的，相关版权由起点中文网出售。第一部卖给中影，第二部卖给了万达。

系列小说版权怎么能只买一部呢？

中影的版权购买行为十分蹊跷，因为天下霸唱说中影只买了第一部的版权。对于「鬼吹灯」这样一个每一部都具有高度开发价值的系列小说来说，单纯只够买一部小说的版权，是不够聪明的商业做法，不利于连续性的商业开发。因此中影购买「鬼吹灯」第一部后立刻开发《鬼吹灯之九重妖塔》的电影，在整体商业开发构想上，可能有些欠缺。

因为对于一个具有连环系列开发价值的 IP 来说，如果只购买其中一部，无论开发的效果如何，电影之后积累的流量都会导入下一步鬼吹灯作品，为他人做嫁衣。这也是影视公司为何在购买系列作品版权开发项目时都会将该IP一系列开发版权都握在自己手中的原因。

正确的买版权姿势当如工夫影业

工夫影业和万达在购买并进行后续鬼吹灯开发时就非常聪明。在推出电影的同时，还策划推出超级季播剧——其实先制作季播剧再推出电影的模式是现在很多网络剧通用的商业模式。对于一个有价值的故事，通过网络剧积累流量和扩大声势，然后再利用电影票房来视线盈利，是一种非常好的商业计划。只有通过这样的计划，公司才能够在IP 价值上获得利润。

特别值得提出的是，万达和工夫在合作拍摄「鬼吹灯」这个事情上最聪明的一点是：他们没有把电影命名为《鬼吹灯》，因为他们只有第二部的版权，所以没有将最大的价值挖掘点放在「鬼吹灯」身上，而是「寻龙诀」身上（当然这也有电影审查的因素考虑）。以「寻龙诀」为基础进行后续的开发、以「寻龙诀」为基础来做电影的营销和发行，使得现在的观众更多的是对「寻龙诀」这个名字印象深刻，对「鬼吹灯」的品牌反而淡化了。这样的方式对电影制作公司和原作者都是有好处的，相当于帮作者新建了一个品牌，用这个品牌来重新运作独立的IP。那么对于鬼吹灯的粉丝来说，「寻龙诀」就是天下霸唱的一个故事，而电影「寻龙诀」的粉丝会认为这就是一个新IP，是围绕鬼吹灯摸金三人组的新故事。那么围绕「寻龙诀」这个IP的开发，万达等寻龙诀制片方都能够沾得收益，并且能够对其他版权开发收取获得利润——这是一个有商业计划的 IP购买方式。

 


天下霸唱介绍表示：「由于《鬼吹灯》大电影开发已经进行了三四年，在长期合作中建立的默契及信任，促使我跟工夫影业签约，并成立天下99公司，按计划应该再写四到八本，但是可能写不了那么多。估计五年之内，可以写三四本。已确定的书名依次为《鬼吹灯摸金校尉》、《鬼吹灯搬山道人》和《鬼吹灯九幽将军》《鬼吹灯阴阳端公》，鬼吹灯摸金三人组的故事，仅在此作品中延续。

天下霸唱表示起点中文网卖的第二部的版权就是万达影业拍摄《鬼吹灯之寻龙诀》，也是正式授权的。

你以为买了一部小说版权就取得 IP 的商业价值了？

非也。甚至很多制片人以为把一个版权做版权登记就获得原创 IP 的商业价值了，naïve。IP 关系着内容、品牌、市场认知度、人物形象及世界观设定，贯穿到内容独创性和很多排他性因素等各个法律层面，是一个靠交易协议织成的法律「锁」。所以很多时候，你以为你获得了版权，可能合同上稍微没有写到，就没有买到这个IP。

也就是说，如果中影在购买第一部的时候，如果没有同时购买第二部，至少要在合同里约定一下对于第二部具有优先购买权，并把优先购买权的行权条件写清楚吧！这样至少还获得了一个未来跟后续公司 PK 的机会，如果做好了，显然能够 PK 掉后者。如果更强势的公司，比如说获取原创 IP（而不是鬼吹灯这么大的 IP）的时候，可以约定作者如果再创作同样世界观（人设、形象、名称等其他）的作品，都要提前告知，并享有优先购买的权利；否则不得为第三方创作这个故事的。

平台卖影视版权存在大问题：雪藏和瞎卖

不过做了那么多版权交易，其实如是君也知道，中影绝对不是故意只买第一部版权的（如果是的话请把这一箭射弯），很有可能是小说版权的独家代理方、网络发布平台起点中文网不想卖的。
 
现在几大主流网络小说平台在与小说作者签约时，已经通过合约取得作者在很长一段时间内（也就是所谓的签约期）创作的小说作品的版权（至少是独家代理权）握在手中，全权的处理小说版权——这个全权是什么意思呢，是说如果平台不帮你推，你的影视版权就卖不出去，如果平台仇恨你，平台就可以把你的版权雪藏了。
 
网文平台除非被比较大的影视机构收购或者有非常紧密的合作关系，否则很难真的推某一部作品的影视版权。所以制片人都要一个一个上去挑，这也是前些日子引发了编剧反 IP大运动一个重要原因。所以平台握着版权，如果有人要买，他们就会卖出去，毕竟网文界和影视圈还是隔层山，所以很多时候网文平台对于影视公司乃至影视的制作团队并没有甄别能力，很可能陷入有钱就卖的地步——这种事情很多嘛，刘慈欣作为作者不也是「没管那么多」，然后把《三体》随便卖了么——后果大家也都知道了咳咳。这个甄别能力可是很专业的能力，其中很重要的就是编剧是谁，所以很多时候超级的网文大大要求自己担任编剧才肯放影视版权……

即使是平台卖版权，作者不应该说「并非我本人授权」

不过还是要吐槽一句，当年《鬼吹灯之九层妖塔》上映的时候，天下霸唱总是说「并非我本人授权」，什么嘛，明明是当年没有跟平台签好合同，搞得平台直接代表作者签合同，所以他除了收到钱可能都没有看中影那个买版权的合同长什么样子……但这不代表不是他本人授权，从法律上讲这就是他本人授权。所以作者大大们请瞪大眼，大神都犯过这样的错误——然而，问题是，难道中影的版权交易协议里没有「作者保证不发表不利于影片的言论」之类的承诺嘛？
